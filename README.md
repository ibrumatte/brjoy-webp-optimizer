# BrJoy Web Optimizer

Pipeline de otimização de imagens para websites.

## Objetivo
Converter JPG/PNG para WebP em lote, mantendo estrutura de diretórios e gerando relatórios de economia.

## Features (V1 - Desktop Pro)
- [ ] Scan recursivo de pastas
- [ ] Manter estrutura de diretórios
- [ ] Presets web (Hero, Thumbnail, Blog, Mobile)
- [ ] Slider de qualidade (60-100%)
- [ ] Relatório de economia (HTML/CSV)
- [ ] Modo não-destrutivo

## Roadmap
- V1: Desktop Pro (conversão em massa + relatórios)
- V2: CLI para CI/CD
- V3: Gerador de snippets `<picture>`

## Instalação
```bash
pip3 install tkinterdnd2
python3 brjoy-converter
```
